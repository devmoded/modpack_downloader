# Об индексе

```toml
[[modpacks]]
name = "<modpack-name>"
version = "<modpack-version>"
source = '<modpack-zip-url>'
```

Выше указаны основные поля, необходимые для загрузчика сборок. Ниже представлено их описание.

### \<modpack-name\>
Название сборки.

**Примеры допустимых названий:**
- "my-pack"
- "my_pack"
- "myPack"
- "MyPack"
- и другие варианты названий без пробелов.

### \<modpack-version\>
Версия сборки.

### \<modpack-zip-url\>
URL `.zip` файла со сборкой, который будет скачан и разархивирован.
